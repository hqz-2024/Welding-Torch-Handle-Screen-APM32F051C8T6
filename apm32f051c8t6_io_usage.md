|        9 | VDDA | VDD_3V3    | 模拟电源接 3.3V。   |

|       48 | VDD  | VDD_3V3    | 数字电源。          |

注意：网表中没有看到 `U4.23` 和 `U4.24`。按 APM32F051 LQFP48 管脚表：

| 物理脚号 | 管脚 | 应连接  |
| -------: | ---- | ------- |
|       23 | VSS  | GND     |
|       24 | VDD  | VDD_3V3 |

需要回到原理图库或原理图确认这两个电源脚是否隐藏连接，或者是否漏接。

## LCD 屏幕接口备注

当前网表中的 LCD 相关连接：

| LCD 相关网络 | MCU 引脚 | 网表连接                                  |
| ------------ | -------- | ----------------------------------------- |
| DC           | PB12     | X1.5                                      |
| SCK          | PB13     | X1.4                                      |
| BLK          | PA5      | 通过 R8/Q2 控制背光，X1.11 为背光相关输出 |
| MOSI         | PB15     | X1.3                                      |
| RES          | PB14     | X1.6                                      |
| VDD_3V3      | -        | X1.10、X1.12                              |
| GND          | -        | X1.7、X1.8、X1.13                         |

需要重点确认：

1. 网表中没有看到 LCD `CS`。若屏幕确实有 CS 脚，则不能悬空。只有一个 SPI 设备时可把 CS 固定接 GND；更推荐接一个 MCU GPIO 便于调试。
2. 当前 X1 是 13P FPC，而之前屏幕规格书文件名为 8pin SPI。必须以实际屏幕 FPC 脚序为准，不能直接套 8pin 模块脚序。

## RS485 / MS2548 接口备注

当前网表中的 MS2548 相关连接：

| MS2548 脚 | 网表网络      | 连接说明                           |
| --------- | ------------- | ---------------------------------- |
| R         | UART1_RX      | 经 R23 到 PB7_RX                   |
| RE        | $1N58         | 与 SHDN 同网，经 R1 上拉到 3.3V    |
| SHDN      | $1N58         | 与 RE 同网，经 R1 上拉到 3.3V      |
| D         | UART1_TX      | 经 R24 到 PB6_TX                   |
| GND       | GND           | 地                                 |
| A         | $1N62 / OUT-A | 经 F2 到 OUT-A，R2 上拉，ESD1 保护 |
| B         | $1N61 / OUT-B | 经 F1 到 OUT-B，R4 下拉，ESD5 保护 |
| VCC       | VDD_3V3       | 当前接 3.3V                        |

MS2548 规格书推荐工作电压为 4.5V 到 5.5V。当前网表给 U1.8 接 3.3V，不在推荐工作范围内，建议确认是否必须改为 5V 供电。若 MS2548 改为 5V 供电，需要确认 PB7_RX 对 5V 接收输出的容忍性，并建议 R25 上拉预留不焊。
